# Task Manager Evolution

This project is part of a series where I explore various aspects of task management applications.

## Overview

The Task Manager Evolution project is built using React and Vite, providing a modern and efficient setup for developing web applications. This project includes features such as:

- Real-time task management
- User authentication
- Responsive design

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone this repository:
```bash
git clone https://github.com/muhammad-husnain-ali-web/task-manager-evolution.git
cd task-manager-evolution/react
```

2. Install the dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173` by default.
